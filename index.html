MDE Optimiser

MDE Optimiser is a tool which can be used to express search and optimisation problems in an MDE Context.

View on Github

What is MDE Optimiser

MDE Optimiser has been developed to allow expressing search and optimisation problems in an MDE context. It considers search spaces to be described by meta-models so that candidate solutions are represented by individual models. Search-space exploration is performed by deriving new models from existing models using Henshin transformations.

MDE Optimiser provides an Xtext based domain-specific language for expressing an optimisation problem.

How to use it

The tool is in early development stage. It has been developed to work as an Eclipse plugin. In order to use it, download or clone the Github repository, import the project in Eclipse, build and run it.
